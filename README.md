## 前置知识

<<(左移运算符) :

> 1 << 1 = 1 * 2^1 = 2;    
> 1 << 2 = 1 * 2^2 = 4;
> 2 << 1 = 2 * 2^1 = 4;
> 2 << 3 = 2 * 2^3 = 16;
> 3 << 10 = 3 * 2^10 = 3072;


<<(左移运算符奇怪的部分):

> 1 << 32 = 1 * 2^(32%32) = 1;
> 2 << 32 = 2 * 2^(32%32) = 2;
> 2 << 33 = 2 * 2^(33%32) = 4;
> 3 << 34 = 2 * 2^(34%32) = 8;
>

(右移运算符)

> 12345 >> 1 = parseInt(12345 / 2^1) = 6172;
> 12345 >> 32 = parseInt(12345 / 2^(32%32)) = 12345;

右移运算符就是用除法取整计算。

& 按位与(and)
